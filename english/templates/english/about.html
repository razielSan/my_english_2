О проекте

Меня зовут Raziel и я являюсь разработчиком данного приложения, которое написано на замечательном веб-фреймворке Flask, языка программирования Python

Приложение создано чтобы показать работу Flask View, используя ORM SQLAlchemy совместно с лёгковесной файловой базой данных SQLite

В качестве css-фреймворка используется модуль bootstrap-flask c версией Bootstrap 5

Что умеет веб-приложение?

  2. Изменять данные
  3. Удалять данные
  4. Просматривать данные
  5. Фильтровать данные
  6. Возможно что-то еще ;)
